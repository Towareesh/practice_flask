# Building

This is a training project on flask.

## Installation

```bash
git clone https://github.com/Towareesh/practice_flask.git
```
Use the package manager [pip](https://pip.pypa.io/en/stable/) to install packages.

```bash
pip install -r requirements.txt
```

## Usage

```cmd
flask shell
```
```flask shell
db.create_all()
db.session.commit()
```
## Launch
```cmd
flask run
```
or
```cmd
python3 runner.py
```
